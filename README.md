### ğŸ›¡ï¸ FakeTLS Obfuscation

Your proxy traffic looks identical to normal HTTPS traffic. The TLS handshake SNI points to a cover domain (e.g., `cloudflare.com`), making it indistinguishable from regular web browsing to any DPI system.

**Traffic masking** goes further â€” when a non-Telegram client probes your server, the connection is forwarded to the real cover domain. Your server responds exactly like cloudflare.com would.

### ğŸ“‹ User Management Recipes

### ğŸ”— Proxy Chaining (Upstream Routing)

Route traffic through intermediate servers:

